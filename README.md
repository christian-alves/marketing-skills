# Marketing Pack — 5 Skills para Claude Code

Pack com 5 skills de marketing para profissionais que usam Claude Code.

## Skills incluídas

| # | Skill | O que faz |
|---|-------|-----------|
| 01 | **Brand Guidelines** | Gera guia completo de identidade visual (cores, fontes, tom de voz) |
| 02 | **Pesquisa com Fontes** | Pesquisa de mercado com dados e referências verificáveis |
| 03 | **Briefing de Campanha** | Cria briefings profissionais com público, canais, cronograma e KPIs |
| 04 | **Carrossel** | Gera carrosséis para Instagram com copy dos slides e legenda |
| 05 | **Sequência de E-mail** | Cria fluxos de e-mail com assuntos otimizados e CTAs estratégicos |

## Como instalar

### 1. Adicione o marketplace

No Claude Code, rode:

```
/plugin marketplace add euchristianalves/marketing-skills
```

### 2. Instale o pack

```
/plugin install marketing-pack@sinoon-marketing
```

Pronto. As 5 skills ficam disponíveis automaticamente.

## Como usar

Depois de instalado, use os comandos:

```
/marketing-pack:brand-guidelines
/marketing-pack:research
/marketing-pack:campaign-brief
/marketing-pack:carousel
/marketing-pack:email-sequence
```

Ou simplesmente descreva o que precisa — o Claude vai ativar a skill certa automaticamente.

## Exemplos

**Gerar brand guidelines:**
```
/marketing-pack:brand-guidelines
> Marca: Café Origem, cafeteria artesanal em SP, público jovem urbano
```

**Pesquisar tendências:**
```
/marketing-pack:research
> Tendências de marketing de conteúdo para e-commerce no Brasil em 2026
```

**Criar briefing:**
```
/marketing-pack:campaign-brief
> Lançamento de curso online de fotografia, orçamento R$ 3.000, 30 dias
```

**Montar carrossel:**
```
/marketing-pack:carousel
> 7 erros que todo empreendedor comete no Instagram
```

**Criar sequência de e-mail:**
```
/marketing-pack:email-sequence
> Sequência de lançamento para SaaS de gestão financeira, 5 e-mails, tom educativo
```

## Requisitos

- [Claude Code](https://claude.ai/download) instalado
- Plano Claude Pro, Max ou Team
